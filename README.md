# Quiz Application
Quiz Application is a responsive web-based project developed using HTML, CSS, JavaScript, and Bootstrap. It allows users to attempt multiple-choice questions in an interactive interface.
The application starts with a start screen and then displays questions one by one with a timer. Users select answers, and the system automatically checks them and updates the score. At the end, the final result is shown, and the user can download it as a PDF.
Key features include a timer for each question, automatic answer handling, progress tracking, score calculation, and dark/light theme support.
This project demonstrates important web development concepts such as DOM manipulation, event handling, timers, and responsive UI design.
